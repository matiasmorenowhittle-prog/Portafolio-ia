# Portafolio IA Aplicada

Proyecto inicial de una ruta de aprendizaje para integrar IA en proyectos de Ingenieria de Sistemas.

## Objetivo

Crear una pagina web simple donde se documenten proyectos, prompts, resultados y aprendizajes relacionados con IA aplicada.

## Como abrir

Abre el archivo `index.html` en tu navegador.

## Que incluye

- Presentacion del objetivo.
- Ruta inicial de proyectos.
- Bitacora del proyecto inicial.
- Recurso visual propio para representar la ruta de aprendizaje.

## Publicacion

El archivo `linkedin_post.md` queda como borrador externo para publicar avances, pero no forma parte visible de la pagina web.

## Siguiente mejora recomendada

Convertir esta pagina en un portafolio dinamico con una lista de proyectos cargada desde un archivo JSON.
